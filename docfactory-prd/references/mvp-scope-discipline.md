# MVP Scope Discipline

Rules:

- MUST list should be the smallest shippable vertical slice.
- SHOULD list must be removable without breaking core loop.
- NICE list is explicitly post-MVP.

Common scope traps:

- Too many screens
- Too many integrations
- Over-engineering for scale
- Premature analytics/experimentation frameworks

If scope feels big:

- Cut features, not quality (keep consistency and states).
