# Monetization Checklist (Freemium + Subscription)

- Define the primary paywall trigger(s)
- Define premium entitlement name(s) (e.g., `pro`)
- Define what is free:
  - limits (credits/day, exports, styles)
- Define what is premium:
  - unlimited usage, higher quality, faster queue, watermark removal
- Restore purchases path
- Trial strategy hypothesis (if uncertain, mark [ASSUMPTION] and validation plan)
