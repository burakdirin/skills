# Component Inventory Guidance (Minimalist iOS-native)

Prefer a small set of consistent primitives:

- Button (4 variants)
- TextField (with validation)
- Card + ListRow
- Modal/BottomSheet (one standard)
- Toast (one)
- EmptyState + Skeleton

Rule of thumb:

- Build features by composing primitives; avoid one-off bespoke components in MVP.
